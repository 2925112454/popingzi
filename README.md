# POPINGZI 破瓶子内容管理系统

## 项目简介

POPINGZI（破瓶子）是一套轻量级的 PHP 内容管理 / 卡片分享网站系统，采用原生 PHP + MySQL 架构，源代码完全开源无加密。系统内置卡片发布、广告管理、用户注册登录、评论互动、投稿审核、轮播图、SEO 优化、社交分享等丰富功能，适合搭建内容分享站、资源导航站、图文卡片社区等。

## 主要功能

- 🎴 **卡片内容管理**：支持多分类卡片发布、编辑、排序、置顶
- 📢 **广告位管理**：内置多处广告位，支持图片广告配置
- 👥 **用户系统**：注册、登录、个人中心、权限分级
- 💬 **评论系统**：用户评论、审核、删除管理
- 📝 **投稿功能**：前端用户投稿，后台审核发布
- 🖼️ **轮播图管理**：首页轮播图后台可视化配置
- 🔍 **SEO 优化**：OG 标签、SiteMap、Robots.txt 自动生成
- 🔗 **社交分享**：支持多平台一键分享
- ⚙️ **系统设置**：站点名称、描述、SMTP 邮件、短信配置等
- 📱 **响应式适配**：兼容 PC 与移动端访问

## 快速开始

### 1. 上传文件

将压缩包内的 **所有文件** 上传至你的网站根目录（或指定运行目录）。

### 2. 创建数据库

在你的 MySQL 中新建一个数据库，编码推荐使用 `utf8mb4`。

根据你的 MySQL 版本，选择对应数据库结构文件导入：
- `[重要]Mysql5.0-8.0(数据库结构文件)/Mysql_5.0版本.sql`
- `[重要]Mysql5.0-8.0(数据库结构文件)/Mysql_5.5版本.sql`
- `[重要]Mysql5.0-8.0(数据库结构文件)/Mysql_5.7版本.sql`
- `[重要]Mysql5.0-8.0(数据库结构文件)/Mysql_8.0版本.sql`

### 3. 修改数据库配置

编辑 `inc/conn.php` 文件，修改以下数据库参数：

```php
$mysql_server_name = '127.0.0.1';  // 数据库服务器地址
$mysql_username    = 'root';       // 数据库用户名
$mysql_password    = '123456';     // 数据库密码
$mysql_database    = 'popingzi';   // 数据库名
```

### 4. 访问安装

配置完成后，访问你的网站域名即可正常使用。

### 5. 默认账号

- **账号**：`12345678`
- **密码**：`admin12345678`

> 🔐 **重要安全提示**：部署上线后 **请务必及时修改默认账号和密码**，防止被恶意登录！
